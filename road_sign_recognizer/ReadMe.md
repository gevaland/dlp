## Использование
Чтобы запустить наш сервис необходимо использовать команды:
```sh
git clone [https://github.com/BekusovMikhail/number_plate.git](https://github.com/gevaland/dlp.git)
cd dlp/road_sign_recognizer
docker compose build
docker compose up
http://127.0.0.1:8501/
```
